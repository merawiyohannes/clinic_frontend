# Clini Desktop

Clini Desktop is a Python Tkinter–based clinic management application designed for small clinics.

It provides a simple desktop interface to manage patients and basic clinic operations locally.

## Features

- Desktop GUI built with Tkinter
- Patient management
- Local database support
- Simple and lightweight workflow
- Designed for small clinics

## Tech Stack

- Python
- Tkinter
- SQLite

## Setup

1. Clone the repository
2. Create virtual environment
3. Install dependencies
4. Run the app
